# Time-Travel
We move through time to see what was, what is, and what is expected to be

## Technological Time Travel - Glimpse of the Future

An interactive web experience built with React.js and Tailwind CSS that allows users to explore different time periods through a beautiful telescope interface.

### Features

- **Telescope Scene**: Navigate through Past (1990), Present (2026), and Future (2040)
- **Year Selector**: Shows only one year at a time, scrolls to reveal others
- **Interactive Controls**: Color-changing arrow buttons aligned horizontally (Past-Left, Present-Middle, Future-Right)
- **Smooth Animations**: Telescope zooms in and moves downward, transitioning to circular layout
- **Circular Slides**: 4 scrollable slides showcasing characteristics of each time period
- **Year Navigation**: Forward/Backward navigation to explore different years with changing background colors
- **Responsive Design**: Optimized for all screen sizes

### Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

### Usage

1. Select a time period using the Past (←), Present (↑), or Future (→) buttons
2. Watch the year selector scroll to show the selected year
3. Click "Take a glimpse" 
4. Navigate through slides using arrow buttons or indicators
5. Use Forward/Backward labels to explore different years (changes background color)
6. Click "off the glimpse" to return to the telescope scene

### Technologies

- **React.js** - Component-based UI framework
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Fast build tool and dev server
- Modern ES6+ JavaScript
